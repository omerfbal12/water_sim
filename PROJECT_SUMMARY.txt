================================================================================
                          GETTING STARTED
================================================================================

PREREQUISITES (macOS):
  1. Install Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Install SDL2: brew install sdl2
  3. Xcode Command Line Tools: xcode-select --install

PREREQUISITES (Linux):
  1. Ubuntu/Debian: sudo apt-get install libsdl2-dev
  2. Fedora: sudo dnf install SDL2-devel
  3. Arch: sudo pacman -S sdl2

BUILD:
  cd /path/to/zed-base
  ./build.sh
  
  (Or use: make  or  mkdir build && cd build && cmake .. && make)

RUN:
  ./splash_sim

================================================================================
                      TROUBLESHOOTING
================================================================================

ISSUE: "SDL2 not found"
SOLUTION (macOS):
  brew install sdl2
  brew link sdl2
  ./build.sh

SOLUTION (Linux):
  Ubuntu/Debian: sudo apt-get install libsdl2-dev
  Fedora: sudo dnf install SDL2-devel
  Arch: sudo pacman -S sdl2
  ./build.sh

ISSUE: "No compiler found"
SOLUTION (macOS):
  xcode-select --install

SOLUTION (Linux):
  sudo apt-get install build-essential

ISSUE: Window doesn't appear
SOLUTION:
  • Check terminal for error messages
  • Ensure SDL2 is properly installed
  • Try: ./splash_sim 2>&1
  • Restart terminal and try again

ISSUE: Build fails with linker errors
SOLUTION:
  make clean
  brew reinstall sdl2  (macOS)
  ./build.sh
